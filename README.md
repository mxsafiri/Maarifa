## Getting Started
1. Clone the repository
```bash
git clone https://github.com/mxsafiri/Maarifa.git
cd Maarifa
```

2. Set up the backend
```bash
cd backend
python -m venv venv
source venv/bin/activate  # On Windows: .\venv\Scripts\activate
pip install -r requirements.txt
```

3. Set up the frontend
```bash
cd frontend
npm install
```
